# Contributing to MediGuard AI

Thank you for your interest in contributing to MediGuard AI! Here are the guidelines to ensure a smooth workflow.

## 🛠️ Local Development Setup

1. Fork the repository on GitHub.
2. Clone your fork locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/medi-guard-ai.git
   cd medi-guard-ai
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the local server:
   ```bash
   npm run dev
   ```

## 🚀 Creating Pull Requests

1. Create a new branch for your feature or bugfix:
   ```bash
   git checkout -b feature/your-feature-name
   ```
2. Make your changes and commit them with descriptive messages.
3. Verify that the application builds successfully:
   ```bash
   npm run build
   ```
4. Push your branch to GitHub and open a Pull Request.

## 📝 Commit Standards

Please keep commit messages concise and descriptive (e.g., `fix: resolve circular risk gauge rendering offset`).
